## 概述

Viabar 帮助你以关键里程碑为核心管理多个项目，而非纠结于流水线式的原子任务。它提供高信息密度的全局看板，一屏聚合所有项目进度与健康度，让你随时掌控全局。

**免费、无广告、Apple 原生体验。要求 macOS 26+。**

## 核心功能

### 里程碑驱动管理
弱化流水线式的原子任务，以关键里程碑为管理节点。辅以轻量级子任务与上下文流式备忘录（Context Memo），降低管理成本，保持项目推进的连续性。

### 极简双栏看板
侧边栏 + 详情区双栏布局，左侧项目列表一目了然，右侧里程碑与备忘录同屏展示。一键切换已完成任务的显隐，灵活掌控信息密度。零学习成本。

### 外延生态常驻（菜单栏 + Widget）
macOS 菜单栏下拉面板提供快捷入口，实时查看当前任务和提醒。搭配桌面 Widget，锁屏或桌面直接展示项目进度条，无需打开主 App。

### 间歇性唤醒提醒
按项目维度设置周期性提醒，触发时自动穿透到当前未完成的顶层任务，告知下一步该做什么，无需重新梳理上下文。

### 模板化初始化
创建不同类型的项目模板，预设里程碑和子任务结构。新建同类项目时，一键从模板生成，自动填充基础框架，即开即用。

### 自动化报告生成
自动提取本周完成、下周待办、本月完成等数据，一键复制粘贴即用。支持周报、月报、下周计划三种报告类型。

### 本地定时备份
多级备份策略：24 小时内每小时、7 天内每天、6 个月内每周。明文 JSON 格式本地存储，数据永不丢失。

## 痛点与方案

| 痛点 | 解决方案 |
|---|---|
| 多项目并行，缺乏全局感 | Dashboard 总览 — 高信息密度全局看板 |
| 原子任务管理成本过高 | 里程碑驱动 — 以里程碑为核心管理节点 |
| 已完成任务一刀切隐藏 | 灵活显隐策略 — 随时切换查看 |
| 功能繁杂，注意力分散 | 双栏极简看板 — 仅保留里程碑与备忘录 |
| 项目搁置后上下文丢失 | 间歇性唤醒提醒 — 告知从哪继续 |
| 细节信息随时间丢失 | 上下文备忘录 — 项目强绑定流式备忘 |
| 同类项目重复初始化 | 模板化初始化 — 一键从模板生成 |
| 查看进度的操作路径太长 | 外延生态常驻 — 菜单栏 + 桌面 Widget |
| 手动拼凑周报月报压力大 | 自动化报告生成 — 一键提取和复制 |
| 历史数据分散不易查找 | 本地定时备份 — 多级策略，明文 JSON |
