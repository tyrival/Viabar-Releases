## Übersicht

Viabar hilft Ihnen, mehrere Projekte zu verwalten, indem es sich auf wichtige Meilensteine statt auf granulare Einzelaufgaben konzentriert. Es bietet ein hochdichtes globales Dashboard, das den gesamten Projektfortschritt und -zustand auf einem einzigen Bildschirm zusammenfasst, sodass Sie stets den Überblick behalten.

**Kostenlos. Werbefrei. Natives Apple-Erlebnis. Erfordert macOS 26+.**

## Funktionen

### Meilensteinorientierte Verwaltung
Konzentrieren Sie sich auf wichtige Meilensteine statt auf pipeline-artige Einzelaufgaben. Unterstützt durch leichtgewichtige Teilaufgaben und kontextbezogene fortlaufende Memos (Context Memo), die den Verwaltungsaufwand reduzieren und gleichzeitig die Projektkontinuität wahren.

### Minimales Zwei-Fenster-Board
Seitenleiste + Detailansicht im Zwei-Fenster-Layout. Links die Projektliste, rechts Meilensteine und Memos gemeinsam angezeigt. Mit einem Klick die Sichtbarkeit abgeschlossener Aufgaben umschalten für flexible Informationsdichte. Keine Einarbeitungszeit.

### Erweitertes Ökosystem (Menüleiste + Widget)
Das Dropdown-Menü in der macOS-Menüleiste bietet einen schnellen Zugang, um aktuelle Aufgaben und Erinnerungen in Echtzeit einzusehen. In Kombination mit Desktop-Widgets sind Projektfortschrittsbalken auf dem Sperrbildschirm oder Desktop sichtbar — die Haupt-App muss nicht geöffnet werden.

### Intervall-Weck-Erinnerungen
Richten Sie periodische Erinnerungen auf Projektebene ein. Bei Auslösung zeigen sie automatisch die nächste unerledigte Aufgabe an und sagen Ihnen genau, was als Nächstes zu tun ist, ohne dass Sie den Kontext rekonstruieren müssen.

### Vorlagenbasierte Initialisierung
Erstellen Sie projektartige Vorlagen mit vordefinierten Meilenstein- und Teilaufgabenstrukturen. Beim Start eines neuen Projekts desselben Typs generieren Sie es mit einem Klick aus der Vorlage — das Grundgerüst wird automatisch ausgefüllt.

### Automatisierte Berichterstellung
Automatisches Extrahieren abgeschlossener Aufgaben (diese Woche), anstehender Aufgaben (nächste Woche) und monatlicher Abschlüsse. Unterstützt Wochen-, Monats- und Planungsberichte für die nächste Woche mit Ein-Klick-Kopie.

### Lokale geplante Backups
Mehrstufige Sicherung: stündlich für 24h, täglich für 7d, wöchentlich für 6mo. Reines JSON-Format, lokal gespeichert. Ihre Daten gehen nie verloren.

## Problemfelder & Lösungen

| Problemfeld | Lösung |
|---|---|
| Fehlende projektübergreifende Übersicht | Dashboard-Übersicht — hochdichtes globales Board |
| Hoher Verwaltungsaufwand | Meilensteinorientiert — Meilensteine verwalten, nicht Einzelaufgaben |
| Abgeschlossene Aufgaben verschwinden | Flexible Sichtbarkeitsumschaltung — jederzeit ein-/ausblenden |
| Überladene Funktionen | Minimales Zwei-Fenster-Board — nur Meilensteine und Memos |
| Kontextverlust bei Projektpausen | Intervall-Weck-Erinnerungen — zeigt den Wiedereinstiegspunkt |
| Details gehen mit der Zeit verloren | Kontextbezogene Memos — projektgebundene fortlaufende Notizen |
| Wiederholte Projekteinrichtung | Vorlagenbasierte Initialisierung — Ein-Klick aus Vorlagen |
| Langer Zugriffsweg zur Fortschrittskontrolle | Erweitertes Ökosystem — Menüleiste + Desktop-Widget |
| Berichtsdruck | Automatisierte Berichte — Ein-Klick-Extraktion und -Kopie |
| Verstreute historische Daten | Lokale geplante Backups — mehrstufig, reines JSON |
