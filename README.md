## Overview

Viabar helps you manage multiple projects by focusing on key milestones rather than granular atomic tasks. It provides a high-density global dashboard that aggregates all project progress and health in a single screen, so you always know where things stand.

**Free. No ads. Native Apple experience. Requires macOS 26+.**

## Features

### Milestone-Driven Management
Focus on key milestones instead of pipeline-style atomic tasks. Supported by lightweight subtasks and contextual streaming memos (Context Memo), reducing management overhead while maintaining project continuity.

### Minimal Dual-Panel Board
Sidebar + detail dual-panel layout. Project list on the left, milestones and memos displayed together on the right. Toggle completed task visibility with one click for flexible information density. Zero learning curve.

### Extended Ecosystem (Menu Bar + Widget)
macOS Menu Bar dropdown provides a quick entry point to view current tasks and reminders in real-time. Paired with desktop Widgets, project progress bars are visible on your lock screen or desktop — no need to open the main app.

### Interval Wake-Up Reminders
Set periodic reminders at the project level. When triggered, they automatically surface the next unfinished task, telling you exactly what to do next without needing to reconstruct context.

### Template-Based Initialization
Create project-type templates with preset milestone and subtask structures. When starting a new project of the same type, generate from the template in one click with the framework auto-filled.

### Automated Report Generation
Auto-extract completed tasks (this week), upcoming todos (next week), and monthly completions. Supports weekly, monthly, and next-week planning reports with one-click copy.

### Local Scheduled Backups
Multi-tier backup: hourly for 24h, daily for 7d, weekly for 6mo. Plain JSON format stored locally. Your data is never lost.

## Pain Points & Solutions

| Pain Point | Solution |
|---|---|
| Lack of overview across projects | Dashboard Overview — high-density global board |
| High management overhead | Milestone-Driven — manage milestones, not atomic tasks |
| Completed tasks disappear | Flexible Visibility Toggle — show/hide anytime |
| Feature bloat | Minimal Dual-Panel Board — only milestones and memos |
| Context loss on project hiatus | Interval Wake-Up Reminders — tells you where to resume |
| Details lost over time | Contextual Memos — project-bound streaming notes |
| Repetitive project setup | Template-Based Initialization — one-click from templates |
| Long access path to check progress | Extended Ecosystem — Menu Bar + Desktop Widget |
| Reporting pressure | Automated Reports — one-click extraction and copy |
| Scattered historical data | Local Scheduled Backups — multi-tier, plain JSON |
