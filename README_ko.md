## 개요

Viabar는 세분화된 개별 작업보다 핵심 마일스톤에 집중하여 여러 프로젝트를 관리할 수 있도록 도와줍니다. 모든 프로젝트의 진행 상황과 상태를 한 화면에 집약한 고밀도 글로벌 대시보드를 제공하므로, 항상 프로젝트 현황을 한눈에 파악할 수 있습니다.

**무료. 광고 없음. 네이티브 Apple 경험. macOS 26+ 필요.**

## 주요 기능

### 마일스톤 중심 관리
파이프라인식 개별 작업 대신 핵심 마일스톤에 집중하세요. 경량 서브태스크와 상황별 스트리밍 메모(Context Memo)를 지원하여 관리 부담을 줄이면서 프로젝트의 연속성을 유지합니다.

### 미니멀 듀얼 패널 보드
사이드바 + 상세 보기 2단 레이아웃. 왼쪽에는 프로젝트 목록, 오른쪽에는 마일스톤과 메모를 함께 표시합니다. 완료된 작업 표시 여부를 클릭 한 번으로 전환하여 정보 밀도를 자유롭게 조정할 수 있습니다. 학습 부담이 전혀 없습니다.

### 확장 생태계 (메뉴 바 + 위젯)
macOS 메뉴 바 드롭다운으로 현재 작업과 알림을 실시간으로 빠르게 확인할 수 있습니다. 데스크톱 위젯과 함께 사용하면 잠금 화면이나 바탕화면에서 프로젝트 진행률을 바로 확인할 수 있어 메인 앱을 열 필요가 없습니다.

### 간격 깨우기 알림
프로젝트 단위로 주기적 알림을 설정하세요. 알림이 트리거되면 다음 미완료 작업을 자동으로 알려주어, 맥락을 다시 파악할 필요 없이 다음에 무엇을 해야 할지 정확히 안내합니다.

### 템플릿 기반 시작
마일스톤과 서브태스크 구조가 미리 설정된 프로젝트 유형 템플릿을 만들 수 있습니다. 동일한 유형의 새 프로젝트를 시작할 때 템플릿에서 원클릭으로 생성하면 프레임워크가 자동으로 채워집니다.

### 자동 보고서 생성
완료된 작업(이번 주), 예정된 할 일(다음 주), 월간 완료 항목을 자동으로 추출합니다. 주간, 월간, 다음 주 계획 보고서를 지원하며 원클릭으로 복사할 수 있습니다.

### 로컬 예약 백업
다중 계층 백업: 24시간 단위, 7일 단위, 6개월 단위. 순수 JSON 형식으로 로컬에 저장됩니다. 데이터가 유실될 걱정이 없습니다.

## 페인 포인트와 해결책

| 페인 포인트 | 해결책 |
|---|---|
| 여러 프로젝트의 전체 현황 파악 어려움 | 대시보드 개요 — 고밀도 글로벌 보드 |
| 높은 관리 부담 | 마일스톤 중심 — 개별 작업이 아닌 마일스톤 관리 |
| 완료된 작업이 사라짐 | 유연한 표시 전환 — 언제든지 표시/숨기기 |
| 기능 과잉 | 미니멀 듀얼 패널 보드 — 마일스톤과 메모만 제공 |
| 프로젝트 중단 시 맥락 상실 | 간격 깨우기 알림 — 어디서 재개할지 안내 |
| 시간 경과에 따른 세부 정보 손실 | 상황별 메모 — 프로젝트에 연결된 스트리밍 노트 |
| 반복적인 프로젝트 설정 | 템플릿 기반 시작 — 템플릿에서 원클릭 생성 |
| 진행 상황 확인까지의 긴 접근 경로 | 확장 생태계 — 메뉴 바 + 데스크톱 위젯 |
| 보고서 작성 부담 | 자동 보고서 — 원클릭 추출 및 복사 |
| 흩어져 있는 이력 데이터 | 로컬 예약 백업 — 다중 계층, 순수 JSON |
