## 概要

Viabar は、細分化されたタスクではなく、主要なマイルストーンに集中することで複数のプロジェクトを管理します。すべてのプロジェクトの進捗状況を 1 画面に集約した高密度なグローバルダッシュボードを提供し、常に全体像を把握できます。

**無料。広告なし。ネイティブな Apple 体験。macOS 26 以降が必要です。**

## 主な機能

### マイルストーン指向の管理
パイプライン方式の細分化されたタスクではなく、主要なマイルストーンに集中。軽量なサブタスクとコンテキストに紐づくメモ（コンテキストメモ）により、管理の手間を抑えつつプロジェクトの継続性を維持します。

### ミニマルなデュアルパネルレイアウト
サイドバー＋詳細の 2 ペインレイアウト。左側にプロジェクト一覧、右側にマイルストーンとメモを同時表示。完了したタスクの表示/非表示をワンクリックで切り替えられ、情報密度を柔軟に調整できます。学習コストはゼロです。

### 拡張エコシステム（メニューバー＋ウィジェット）
macOS メニューバーのドロップダウンから、現在のタスクとリマインダーをリアルタイムに確認できるクイックアクセス機能。デスクトップウィジェットと組み合わせることで、ロック画面やデスクトップ上にプロジェクトの進捗バーを表示でき、アプリを開く必要はありません。

### インターバル通知リマインダー
プロジェクトレベルで定期的なリマインダーを設定。トリガー時に次の未完了タスクを自動的に特定して通知するため、コンテキストを再構築する手間なく「次に何をすべきか」が明確になります。

### テンプレートによる初期化
マイルストーンとサブタスクの構成を事前定義したプロジェクトテンプレートを作成。同じタイプの新規プロジェクトを開始する際、ワンクリックでテンプレートから生成し、骨組みを自動入力できます。

### レポート自動生成
今週の完了タスク、来週の予定タスク、月間完了実績を自動抽出。週次・月次・来週計画のレポートに対応し、ワンクリックでコピーできます。

### ローカルスケジュールバックアップ
多層バックアップ：24 時間は毎時、7 日間は毎日、6 ヶ月間は毎週。プレーンな JSON 形式でローカルに保存。データが失われることはありません。

## 課題と解決策

| 課題 | 解決策 |
|---|---|
| プロジェクト全体の概要が見えない | ダッシュボード概要 — 高密度なグローバルボード |
| 管理の手間が大きい | マイルストーン指向 — 細かいタスクではなくマイルストーンを管理 |
| 完了したタスクが見えなくなる | 柔軟な表示切替 — いつでも表示/非表示を切替可能 |
| 機能が多すぎる | ミニマルなデュアルパネル — マイルストーンとメモのみ |
| プロジェクト中断時のコンテキスト喪失 | インターバル通知 — どこから再開すべきかを通知 |
| 時間経過による詳細の忘却 | コンテキストメモ — プロジェクトに紐づく継続的なメモ |
| プロジェクト立ち上げの手間 | テンプレート初期化 — テンプレートからワンクリックで作成 |
| 進捗確認までのアクセス手順が長い | 拡張エコシステム — メニューバー＋デスクトップウィジェット |
| レポート作成の負担 | 自動レポート — ワンクリックで抽出・コピー |
| 過去データの散在 | ローカルスケジュールバックアップ — 多層・プレーン JSON |
